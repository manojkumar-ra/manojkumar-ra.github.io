FineTuneX
LLM Fine-Tuning

Full-stack platform to upload custom datasets and fine-tune GPT-2 using LoRA adapters, training only 1.4% of 124M parameters. Features a real-time loss curve, AI-powered analysis, and a chat interface comparing fine-tuned vs base model outputs.

Python PyTorch HuggingFace PEFT/LoRA FastAPI MySQL Groq API
  • LoRA fine-tuning with configurable rank, alpha, and epochs
  • Real-time training loss visualization with Chart.js
  • Side-by-side chat comparison (fine-tuned vs base model)
  • AI-powered training result analysis using Llama 3.3 70B

AgentFlow
AI Agents + RAG

Autonomous AI agent powered by LLaMA 3.3 70B that decomposes tasks into steps and dynamically selects from 6 tools. Features a RAG pipeline with ChromaDB for document Q&A and real-time reasoning streamed via SSE.

Python FastAPI Groq API ChromaDB Sentence Transformers MySQL
  • 6 tools: web search, Python exec, calculator, Wikipedia, datetime, RAG
  • RAG pipeline with PDF, TXT, DOCX upload and vector search
  • Real-time agent reasoning via Server-Sent Events

AnomalyX
Anomaly Detection

Multi-method anomaly detection system combining Isolation Forest, Autoencoder (PyTorch), and Z-Score analysis with a weighted voting mechanism. Features interactive visualizations and AI-powered explanations.

Python PyTorch scikit-learn SciPy FastAPI MySQL
  • 3 detection methods with weighted voting (2+ agree = anomaly)
  • Interactive scatter plots and score distribution charts
  • Customizable contamination rates and column selection
  • AI explanations using Groq (Llama 3.3 70B)
